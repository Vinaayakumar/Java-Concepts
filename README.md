# vincloud Application

## Description
This is a Spring Boot application integrated with AngularJS.

## Setup Instructions
1. Clone the repository:
    ```bash
    git clone 
    ```

2. Navigate to the project directory:
    ```bash
    cd vincloud
    ```

3. Build the project using Maven:
    ```bash
    mvn clean install
    ```

4. Run the application:
    ```bash
    mvn spring-boot:run
    ```

## Dependencies
- Spring Boot
- AngularJS
